# Pneumonia Detection Using CNN

## Project Description
This project detects Pneumonia from Chest X-ray images using a Convolutional Neural Network (CNN).
Grad-CAM is used to highlight abnormal lung regions.

## Technologies Used
- Python
- TensorFlow
- Keras
- Flask
- OpenCV

## Features
- Upload chest X-ray image
- Predict Normal or Pneumonia
- Display confidence score
- Show heatmap (for pneumonia cases)
- Downloadable medical report

## How to Run
1. Install requirements: pip install -r requirements.txt
2. Run: python app.py
3. Open browser and upload image
